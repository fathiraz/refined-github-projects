# Privacy Policy

**Refined GitHub Projects** is a browser extension that adds bulk-action capabilities to the GitHub Projects V2 table view.

## What we collect

Nothing. The extension collects **zero** user data.

## Your Personal Access Token (PAT)

- Your PAT is stored **exclusively** in `chrome.storage.local` on your own device.
- It is **never** transmitted to any server other than `api.github.com`.
- There is **no proxy**, **no relay**, and **no third-party service** involved.

## API Calls

All requests go directly from your browser to `https://api.github.com/graphql`. No intermediate server is used.

## Analytics & Telemetry

There are **none**. No analytics, no crash reporting, no usage telemetry of any kind.

## Auditability

This extension is fully open source. You can audit every network request and storage operation in the source code.
